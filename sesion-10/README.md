### data

contexto físico: sala de clases

contexto temporal: proponemos que la primera sesión de consejo de curso de cada mes, esté destinada a juegos. La propuesta contempla que hayan varios juegos disponibles, y que se asigne aleatoriamente a quién le toca este juego. Le profesore tiene la posibilidad de sugerir o decidir quiénes participan en el juego, siempre y cuando, no se repitan 2 semanas seguidas.

cantidad de jugadores: 4

duración: 20-30 minutos

roles: les participantes tienen roles horizontales, entre elles deberán organizar tareas para completar el juego.

cómo se juega: los 4 participantes sentados alrededor de el juego, ya sea en una mesa o en el piso.
